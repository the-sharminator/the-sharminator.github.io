Bootcamp 2021: Zenga

For my Bootcamp project I decided to try and recreate the game Jenga in processing. I chose this project because I thought it would present some interesting problems which would require creative solutions, in particular, working out the physics of whether or not the tower should collapse at any given point. It also provided a good opportunity to practice using Classes, Arrays, and Loops. The project is split into two parts: a simple menu screen, and the game itself. The menu screen displays the controls for the game - we can use the WASD keys to move around, f to pick up or place a block, r to reset the tower, and space to open or close the menu. I've tried to keep the actual game as simple and intuitive as possible. The currently selected block is highlighted in white. Pressing f will pick it up, after which you can move it to a new position, and press f again to place it. Each time you pick up or place a block, the game will calculate whether or not the tower should collapse. When it does, you can simply press r to reset it and start again.
